开源版 LIGHT CHASER 是一款基于 React18、Vite5、TypeScript5 的数据可视化设计工具，面向大屏展示、数据报告、数据分析等场景，提供高性能的可视化搭建能力。

## 开源版定位

- 面向个人开发者、学习者和团队内部可视化项目
- 提供完整的可视化编辑器与基础发布能力
- 支持常见数据源与图表扩展，适合二次开发

## 开源版核心能力

- 拖拽式设计：组件位置和尺寸支持可视化拖拽
- 高性能渲染：组件独立渲染，可承载 1000+ 组件同屏
- 丰富数据源：MySQL、Oracle、PostgreSQL、SQLServer、API、WebSocket、MQTT 等
- 图表扩展：支持 G2Plot、Echarts 自定义组件
- 蓝图交互：支持节点化交互编排与数据联动
- 跨平台：支持 Windows、Linux、Mac 及现代浏览器

## 推荐：LIGHT CHASER Pro（私有化部署）

如果你正在构建企业级数据可视化平台，推荐使用 Pro 云端版，获得更完整的协作与管理能力。

- 官网入口：https://lcpdesigner.cn/home
- 适用场景：团队协作、权限管理、业务化交付
- 推荐人群：商业项目、生产环境、需要持续扩展的团队

## 推荐：LIGHT CHASER Pro 桌面端

如果你希望本地化部署体验、离线使用或快速交付单机方案，推荐使用 Pro 桌面端。

- 下载入口：https://lcpdesigner.cn/download
- 适用场景：离线演示、内网环境、轻量部署
- 推荐人群：需要开箱即用桌面客户端的个人或团队
