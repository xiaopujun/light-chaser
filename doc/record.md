# 本地背景图片预览

本地背景图片的生成，可以使用两种方式。

1. 将上传的图片转换为base64格式，然后将base64格式的图片作为背景图片。（性能很差）
2. 将上传的图片保存到indexedDB中，使用URL.createObjectURL()方法生成图片的URL，然后将图片的URL作为背景图片。（性能较好）
